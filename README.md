# Sonik

[WIP] Server-Side-Only web framework.

## Quick Start

Just try it.

```
npx degit yusukebe/sonik/examples/basic my-app
cd my-app
```

npm:

```
npm install
npm run dev
npm run build
npm run deploy
```

yarn:

```
yarn install
yarn dev
yarn build
yarn deploy
```
